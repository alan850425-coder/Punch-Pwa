# 出勤打卡 PWA

澤天健康事業員工個人出勤打卡系統。

## 功能
- 第一次開啟輸入姓名，之後固定
- 上班 / 下班立即打卡
- 補打卡（選擇任意日期時間）
- 匯出 Excel（含姓名、日期、類型、時間、是否補打）
- 離線可用（PWA）
- 資料存在手機本地，各員工自己管自己

## 部署方式（Vercel）

1. 把這個資料夾推上 GitHub（新建 repo）
2. 到 [vercel.com](https://vercel.com) → Import Git Repository
3. 選這個 repo，Framework Preset 選 **Other**
4. Deploy → 完成，取得網址

## 使用方式

員工用手機瀏覽器開啟網址後：
- iOS：Safari → 分享 → 加入主畫面
- Android：Chrome → 三點選單 → 加入主畫面

之後就像 App 一樣從桌面開啟。
